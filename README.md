# T-Shirt Project

A full-stack MERN application for browsing and managing T-shirts.

## Features
- User authentication
- Product listing
- Add to cart
- Responsive UI
- Backend API integration

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- CSS

## Installation

### Frontend
cd frontend
npm install
npm start

### Backend
cd backend
npm install
npm start
